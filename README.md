PW
==

PW is a personal, GPG-based, password manager.

### Usage

    # Get a prompt to add a new password named google.com.
    $ pw add google.com
   
    # Print google.com password to stdout.
    $ pw get google.com

    # List all passwords
    $ pw list
    
Use `pw -h` to see all available options.
